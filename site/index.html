Reference architecture

Engineering decisions

Production-minded by default.

01

Security

Public access blocked, encrypted storage, least-privilege origin policy, HTTPS redirect, and TLS 1.2 minimum.

02

Performance

Brotli/Gzip compression and distinct cache policies for immutable assets and HTML content.

03

Reliability

Versioned infrastructure, deterministic deployments, SPA-friendly error responses, and safe invalidation.

04

Operations

Remote-ready Terraform state, deployment outputs, validation checks, and a documented rollback path.
